#  Personal Linktree Clone - DevHandal 2026

Hi! I'm **Alfi Febriawan**, a Fullstack Developer. This project is a personalized link directory (Linktree clone) built as part of my portfolio and the **DevHandal 2026** scholarship program.

##  Features
* **Real-time Database**: Integration with **Supabase** for dynamic link management.
* **Dark Mode Design**: Elegant and professional dark UI.
* **Smooth Animations**: Interactive hover effects and fade-in loading animations.
* **Responsive Layout**: Optimized for both mobile and desktop views.
* **Dynamic Icons**: Supports both Emojis and Custom Image Icons for branding.

##  Tech Stack
* **Frontend**: React.js & Vite.
* **Backend/Database**: Supabase.
* **Styling**: Custom CSS3.
* **Deployment**: Vercel.
